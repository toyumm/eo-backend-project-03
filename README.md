# 📋 게시판 프로젝트 (eo-backend-project-03)

스프링 부트와 JPA를 활용한 게시판 애플리케이션 개발

## 🛠️ 기술 스택
* **Language**: Java 17 (ms-17)
* **Framework**: Spring Boot 3.x
* **Database**: MySQL 8.0
* **ORM**: Spring Data JPA
* **View**: Thymeleaf, CSS

## 📌 주요 구현 기능

### 🔐 접근 관리
- [ ] 로그인 및 로그아웃 기능

### 📝 게시글 관리 (Board)
- [ ] 게시글 목록 조회 (페이징 예정)
- [ ] 게시글 상세 열람
- [ ] 게시글 작성 / 수정 / 삭제

### 💬 댓글 관리 (Comment)
- [ ] 댓글 작성
- [ ] 댓글 수정
- [ ] 댓글 삭제

## ⚙️ 개발 환경 설정
* 모든 비밀번호 및 민감 정보는 환경 변수로 관리하여 보안을 유지합니다.
